### how to compile

```
ghc first_prog.hs
./first_prog
```

with changing compile file's name
```
ghc hello.hs -o helloword
```

### enter to ghci 

enter ghci
```
ghci
```

leaving ghci
```
:q
```

load file inside ghci .. and run function inside file
```
GHCI> :l hello.hs
GHCI> main
GHCI> hello "mo"
```
